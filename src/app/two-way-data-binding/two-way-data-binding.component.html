Two way data binding

Two way data binding is used when the data flow is required to be bidirectional
That is data should flow from Model to View and from View to Model.

The syntax used by the two way data binding is the banana in the box syntax [ ( ) ]

It uses the ngModel directive

The in the input box is populated with the data from the model by default
and when it is changed from the view
Since the data flow is bidirectional, the data in the model will also be changed


The value of the variable username in Model is : {{username}}
