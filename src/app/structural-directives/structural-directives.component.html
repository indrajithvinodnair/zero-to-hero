Structural Directives

Structural directives are used to change the DOM layout by either removing or adding elements. These directives have the * symbol as prefix.

The *ngIf, *ngFor,*ngSwitch are the examples of this directive.

Interact with the radio buttons to render an element conditionally using *ngIF


The content inside this content is visible only when the radio button is on the on state.

This is made possible using *ngIf

Other commonly used structual-directives are the *ngFor which can be used to render data on View as table rows or as div elements and *ngSwitch which
is used to check the value of a varible inside a switch statement and render DOM elements accordingly.
